## Twitter Clone (Client)

### Tech Stack -

- Node.js
- GraphQL
- Apollo
- Prisma ORM
- PostgreSQL
- Supabase
- Next JS
- Tailwind CSS
- Shadcn UI (component library)
- TypeScript
- AWS

#### Steps to run this project

- Clone the project
- Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

- Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
